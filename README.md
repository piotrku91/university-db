## How to compile and run,

>   Depedencies:

    - Download file crow_all.h file from official crow repo -> https://github.com/ipkn/crow
    - Install all depedencies from crow repo (check crow manual page)

>   Type:

    cd build
    cmake ..
    make
    ./university-db (for app)
    ctest -V (for tests)
